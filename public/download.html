🔷 鸿蒙安装方法

鸿蒙系统通过 PWA 方式安装，与原生 App 体验一致：

1鸿蒙手机打开 浏览器，访问 http://123.207.204.92:2121
2点击浏览器底部菜单 「⋮」 → 「添加至桌面」
3桌面出现「万年历」图标，点击启动（全屏，无浏览器边框）

如需原生 HAP 包（DevEco Studio 构建），下载 鸿蒙源码包

🔧 源码构建说明

Android: 解压 → Android Studio 打开 → Sync Gradle → Build → Build APK
iOS: 解压 → Xcode 打开 App.xcworkspace → 配置证书 → Product → Archive
鸿蒙: 解压 → DevEco Studio 打开 → Build → Build HAP
前置: 源码包解压后执行 npm install && npx cap sync 同步Web资源
